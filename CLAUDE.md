# MOINK — Chinese Learning Platform

## Project Overview
A full-platform Chinese learning web app for HSK 1–4 learners combining structured vocabulary study with Chinese culture exploration. Fun, motivated, and visually engaging.

## Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- Framer Motion
- Firebase Auth + Firestore (user data)
- Supabase (vocabulary/content database)
- Hanzi Writer (character stroke animation + component data)

## Design Direction
Clean minimal UI with warm cultural personality layered on top.
- Background: white / off-white
- Accent: red (#dc2626)
- Ink black: (#111111)
- Fun, approachable — not childish
- Mascot: red panda named Moink
- Subtle cultural touches (ink, brush strokes) without overdoing it

## Features
- Vocabulary with component similarity grouping (words grouped by shared radicals)
- HSK level filtering (1–4)
- Character decomposition cards with stroke animation
- Quizzes (flashcards, multiple choice)
- Culture section (destinations, food, festivals) linked to vocabulary
- User profile, progress tracking, streaks, bookmarks
- Authentication via Firebase (Google + email)

## Target Users
HSK 1–4 learners, beginner to intermediate Chinese learners globally.

## Monetization
Free for now.
